# IDR-IIsim: Software Documentation

IDR-IISIM is an open-source compiler designed to process industrial models described in YAML format and generate Python scripts that can be integrated into large-scale energy system models and circular economy assessments.

This repository provides a reproducible and modular framework for describing industrial processes as data models, validating them, and generating executable scripts.
